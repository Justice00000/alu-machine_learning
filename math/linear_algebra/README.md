lice Me Up

## Description
This project demonstrates the use of Python's list slicing to extract specific elements from a list. The program is designed to:
- Extract the first two numbers of the list.
- Extract the last five numbers of the list.
- Extract the 2nd through 6th numbers of the list.

The program achieves this without the use of loops or conditional statements and is exactly 8 lines long.

## Files
- `0-slice_me_up.py`: The main Python script that performs the slicing and prints the results.
